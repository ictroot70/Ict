## Финальный smoke по ТЗ

### UC-1 Заполнение/редактирование профиля

- [ ] Обязательные поля валидируются (username/firstName/lastName)
- [ ] Кнопка `Save Changes` неактивна при невалидной форме
- [ ] Дата рождения выбирается через calendar
- [ ] Возраст `<13` показывает сообщение с переходом в Privacy Policy
- [ ] Успешное сохранение показывает `Your settings are saved!`
- [ ] Ошибка сервера показывает `Error! Server is not available!`

### UC-2 Загрузка/удаление фото

- [ ] Выбор фото с компьютера работает
- [ ] JPEG/PNG и лимит 10 MB отрабатывают корректно
- [ ] Кроп/превью работают
- [ ] Закрытие модалки отменяет загрузку
- [ ] Удаление фото: confirm `Yes/No` работает по ТЗ

### UC-3 Редактирование личных данных

- [ ] Переход в `Profile Settings` с профиля работает
- [ ] Изменения сохраняются и пользователь остается на странице редактирования
- [ ] Возврат на профиль без сохранения не применяет несохранённые изменения
